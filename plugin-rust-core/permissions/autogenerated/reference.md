## Default Permission

Default permissions for the any-sync plugin

#### This default permission set includes the following:

- `allow-command`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`any-sync:allow-command`

</td>
<td>

Enables the command command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`any-sync:deny-command`

</td>
<td>

</td>
</tr>
</table>
